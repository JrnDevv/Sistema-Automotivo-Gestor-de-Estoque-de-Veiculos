# Sistema Automotivo – Gestão de Estoque de Veículos

Repositório para o projeto **Sistema Automotivo**, que tem como objetivo gerenciar o estoque de veículos de uma concessionária, permitindo cadastro, edição, exclusão e listagem de veículos.

---

## Estrutura do Projeto

- **Back-end/Java**: Contém a aplicação em Spring Boot, com todas as classes, controllers, repositórios e configurações necessárias para o funcionamento do sistema.
- **Front-end**: Contém os arquivos HTML, CSS e JS que compõem a interface de usuário do sistema.
- **Database**: Scripts e arquivos relacionados ao banco de dados utilizados pela aplicação.
- **.idea**: Configurações do IntelliJ IDEA (opcional para compartilhamento, pode ser ignorado).
- **README.md**: Documentação do projeto.

---

## Funcionalidades

- Cadastro de veículos com informações detalhadas:
  - Marca
  - Modelo
  - Ano
  - Cor
  - Quilometragem
  - Preço
  - Status
- Listagem de veículos com filtros e ordenação.
- Edição e exclusão de veículos existentes.
- Integração entre front-end e back-end.
- Persistência de dados em banco de dados relacional.

---

## Tecnologias Utilizadas

- **Back-end**: Java 17+, Spring Boot, Spring Data JPA
- **Front-end**: HTML5, CSS3, JavaScript
- **Banco de Dados**: H2 / MySQL (dependendo da configuração)
- **Controle de Versão**: Git e GitHub
